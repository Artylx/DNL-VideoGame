# DNL-VideoGame

### Description
Projet de jeu pour un concours de DNL sur l'ANZAC.
**Type:** Jeu 2D
**Langage:** JS, CSS, HTML

## Scénario

### 1ère partie
Le player commence dans un amphithéâtre avec comme objectif de répondre correctement au question. Quand il aura répondu correctement il passe la la deuxième partie.

### 3ère partie
Le player se trouve chez lui avec ses parents. Il va avoir une autre série de questions et si il réussi c'est la victoire.
